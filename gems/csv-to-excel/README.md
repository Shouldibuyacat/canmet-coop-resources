# csv-xlsx-converter Instructions
## Install:
```
gem install csv-xlsx-converter
```
## CSV to Excel xlsx:
```
csv_to_xlsx input.csv
csv_to_xlsx input.csv output.xlsx
```
## Excel xlsx to CSV:
```
csv_to_xlsx input.xlsx
csv_to_xlsx input.xlsx output.csv
```
